MantellaIconSelector demo
Select icon:
Features
  • Compact block of content
  • Wrap on standart select
  • Expanded window for selection
  • Multi columns
  • Displaying icon's titles
Getting started
Insert in head
<!-- basic style -->
<link rel="stylesheet" type="text/css" href="your_path/css/miconselector.css" />
<!-- required javascripts -->
<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.8.1/jquery.min.js"></script>
<script type="text/javascript" src="your_path/js/miconselector.js"></script>
Insert in body
<select name"my_icons"> window content
  <option value="1" image="1.png">title 1</option>
  <option value="2" image="2.png">title 2</option>
  <option value="3" image="3.png">title 3</option>
  <option value="4" image="4.png">title 4</option>
  <option value="5" image="5.png">title 5</option>
  <option value="6" image="6.png">title 6</option>
</select>

<script type="text/javascript">
$(document).ready( function(){
    $('select[name="my_icons"]').mIconSelector({
      	path	: './img/',
      	width	: 24,
      	height	: 24,
      	columns	: 3
	});
});
</script>
Documentation
Options
pathType: string, Default: "./icons/" Path to map of icon-images. widthType: string, Default: 32 Width of icon-image. heightType: string, Default: 32 Height of icon-image. columnsType: integer, Default: 3 Count of columns in popup selection list.
